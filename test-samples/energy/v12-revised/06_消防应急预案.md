# 消防应急预案

## 1. 编制依据
- 《中华人民共和国消防法》
- 《生产安全事故应急预案管理办法》(应急管理部令第2号)
- GB/T 29639-2020《生产经营单位生产安全事故应急预案编制导则》

## 2. 应急组织架构

```
应急指挥部（总指挥：总经理）
  ├── 现场指挥组（安全总监）
  ├── 灭火救援组（消防队长 + 义务消防员 20人）
  ├── 疏散引导组（行政部长 + 各区域安全员）
  ├── 医疗救护组（医务室 + 120联动）
  ├── 通讯联络组（办公室主任）
  └── 后勤保障组（后勤部长）
```

## 3. 重点防火区域

| 区域 | 火灾危险等级 | 灭火剂类型 | 报警系统 |
|------|------------|-----------|---------|
| 甲类仓库 | 甲 | 泡沫/干粉 | 自动感烟+手报 |
| 生产车间 | 乙 | 干粉/CO2 | 自动感温+手报 |
| 配电房 | 丙 | 气体灭火(七氟丙烷) | 自动感烟+手报 |
| 办公楼 | 丙 | 水/干粉 | 自动感烟+手报 |
| 罐区 | 甲 | 泡沫炮 | 可燃气体+手报 |

## 4. 应急响应流程

### 4.1 报警
1. 发现火情立即拨打内部报警电话 119/8119
2. 报告内容：火情位置、燃烧物质、火势大小、有无人员被困
3. 消防控制室确认后启动预案

### 4.2 初期扑救（发现后3分钟内）
1. 使用就近灭火器进行初期扑救
2. 关闭火场区域阀门和电源
3. 如火势无法控制，立即撤离并等待专业队伍

### 4.3 人员疏散
- 疏散路线：按安全出口标识→集合点
- 集合点：厂区东侧停车场
- 清点人数：各部门安全员负责

## 5. 演练计划
- 综合消防演练：每年6月（安全生产月）
- 专项演练：每季度一次
- 桌面推演：每半年一次
- 演练后48小时内完成评估报告
