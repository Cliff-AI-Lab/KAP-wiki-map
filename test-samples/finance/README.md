# 金融行业测试样例（5 文件）

> 导航：[← 测试样例总入口](../README.md) · [← 项目首页](../../README.md) · [决策书 §1.3 第二批扩展](../../docs/01-技术决策书.md)

金融**不在 KAP 首批锁定行业**（[决策书 §1.3](../../docs/01-技术决策书.md) 第二批扩展，GA 后启动）。
本测试集仅作为**跨行业兼容性测试**与未来 GA 后扩展的参考样本，
M0-M5 阶段不投入做金融专属模板。

## 与决策书的关系

[决策书 §1.3](../../docs/01-技术决策书.md) 行业范围：

> **第二批扩展（GA 后）**：金融、政企、医疗、运营商、军工

意味着金融测试样例**目前仅用于"反向证明 KAP 不会强行套用其他行业模板"**，
不参与 M0-M5 任何工位的回归测试基线。
