## 全景成果

### M6 #1 · `as_of` 时光机查询（决策书 §5.3 历史回溯）
ShadowGraphStore 给实体 / 关系加 `created_at` 字段（首次写入盖戳，合并 doc_ids 不覆盖）；旧数据无戳视作"任何时刻可见"，零侵入兼容。
- `entities_as_of(project, version, before)` / `relations_as_of(...)`
- API `GET /api/v1/rebuild/as-of?project_id=&version=&before=ISO8601`
- **关联代码**：[`packages/rebuild/shadow_graph.py`](../../backend/packages/rebuild/shadow_graph.py) + [`api/routers/rebuild.py`](../../backend/api/routers/rebuild.py)

### M6 #2 · 批量观察期 tick-all 端点（外部 cron / ISS-Job 入口）
M5 #2 留的"接定时器"收口。设计原则：不接 Quartz 框架（轻量），任何外部 cron / ISS-Job 走 HTTP 即可。
- `tick_all_observations()` — 单 project 异常静默吞掉，不阻断其他
- API `POST /api/v1/rebuild/observations/tick-all`（RequireRole SME）
- **关联代码**：[`packages/rebuild/promotion_observer.py`](../../backend/packages/rebuild/promotion_observer.py)

### M6 #3 · SME 决策计数 + 简单指标聚合（决策书 §5.3 运营观察）
新增 `packages.observability.decision_log` 函数式模块。
- 7 类 DecisionType：approve/reject_proposal · promote · rollback · alert_acknowledged · facet_approved/rejected
- 派生指标：`approval_rate` / `promote_rollback_ratio`
- 接线：ontology.approve/reject + rebuild.promote/rollback 全部加 hook
- API `GET /api/v1/observability/decisions` + `/aggregate`
- list_decisions 用 reversed 而非 sort（datetime.now 分辨率不足时 sort 失序）
- **关联代码**：[`packages/observability/decision_log.py`](../../backend/packages/observability/decision_log.py) + [`api/routers/observability.py`](../../backend/api/routers/observability.py)
